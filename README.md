# Mathmania

Mathmania is an ios/android application ....

## Installation

Install node and node package manager from here
[node](https://nodejs.org/en/).

Check node version
```bash
node -v
```
and npm version
```bash
npm -v
```

## Usage

To run the application

```bash
npm install
```

```js
ionic serve
```
